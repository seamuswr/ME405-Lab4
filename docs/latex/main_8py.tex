\doxysubsection{Detailed Description}
This file contains a demonstration program that runs some tasks, an inter-\/task shared variable, and a queue. 

This file runs the control system on the microcontroller.

The tasks don\textquotesingle{}t really {\bfseries{do}} anything; the example just shows how these elements are created and run.
